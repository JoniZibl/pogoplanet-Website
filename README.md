# POGO PLANET Website Refactor

This project has been professionals refactored to separate HTML, CSS, and JavaScript.

## Directory Structure

- **src/**: All source code.
  - **pages/**: HTML files (`index.html`, `news.html`, `legal.html`).
  - **styles/**: CSS files (`main.css`, `components/`, `pages/`).
  - **scripts/**: JavaScript files (`main.js`, `home.js`).
  - **assets/**: Images, fonts, and videos.

## How to Run

Simply open `src/pages/index.html` in your web browser.

## Key Changes

- **Separation of Concerns**: HTML no longer contains inline styles or scripts.
- **Organization**: Valid directory structure for scalability.
- **Cleanup**: Removed redundant code and files.
